## Description
I have implemented tic-tac-toe game using react developed by facebook. Tic-tac-toe (American English), noughts and crosses (British English), or Xs and Os is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Requirements and Execution:

Linux User:<br>
All you need to installed node and git in your system.<br>
open terminal and copy paste this line : ``` git clone https://github.com/ssatyamchauhan/React-TicTacToe.git ```<br>
and then type ```npm install``` this will install all the packages required to play this game.<br>
To start the game type command ```npm start``` and then go to you browser and type ```http://localhost:3000```<br>

Enjoy the Game.
